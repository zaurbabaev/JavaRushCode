
Взаимная блокировка
1. Разберись, как работает программа.
2. Не меняя классы Т1 и Т2 сделай так, чтобы их нити завершились, не обязательно успешно.
3. Метод sleeр не использовать.

Требования:
1. Метод main должен запускать нить 1.
2. Метод main должен запускать нить 2.
3. Класс T1 не изменять.
4. Класс T2 не изменять.
5. Метод sleeр не использовать.
6. Вывод программы должен состоять из 2х строк, информирующих о завершении нитей. Например: "T1 was interrupted" и "T2 finished".
7. Нити t1 и t2 должны завершаться (не обязательно успешно).
